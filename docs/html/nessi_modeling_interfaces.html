nessi.swm¶

dispersion_curve_init(verbose)[source]¶

Initialize the dispersion curve calculation.

Parameters:verbose – integer, 0 minimal ouput, 1 verbose output
dispersion_curve_rayleigh(nLayers, h, vp, vs, rho, nSamples, omega, nModes, slowness, group)[source]¶

Calculate the Rayleigh theoretical dispersion curve.

Parameters:
  • nLayers – integer, number of layers
  • h – double, thickness of layers (m)
  • vp – double, P-wave velocity in each layer (m/s)
  • vs – double, S-wave velocity in each layer (m/s)
  • rho – double, density in each layer (kg/m3)
  • nSamples – integer, number of frequency samples
  • omega – double, angular frequencies (rad/s)
  • nModes – integer, number of modes including fundamental
  • slowness – double, output of slowness values
  • group – integer, 0 for phase, 1 for group
