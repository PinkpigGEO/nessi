image(key='tracl', bclip=None, wclip=None, clip=None, legend=0, label1=' ', label2=' ', title=' ', cmap='gray')[source]¶

matplotlib.pyplot.imshow adapted to plot SU files

Parameters:
  • key – header keyword (default tracl)
  • bclip – data values outside of [bclip,wclip] are clipped
  • wclip – data values outside of [bclip,wclip] are clipped
  • clip – clip used to determine bclip and wclip
  • legend – colorbar 0=no colorbar (default) 1=colorbar
  • label1 – x-axis label
  • label2 – y-axis label
  • title – title of the image
  • cmap – color map (defautl ‘gray’): gray, jet, …
kill(key=' ', a=1, min=0, count=1)[source]¶

Zero out traces. If min= is set it overrides selecting traces by header.

Parameters:
  • key – SU header keyword
  • a – header value identifying traces to kill
  • min – first trace to kill
  • count – number of traces to kill
masw(vmin=0.0, vmax=1000.0, dv=5.0, fmin=1.0, fmax=100.0)[source]¶

Calculate the dispersion diagram using MASW method. Return the dispersion diagram, the velocity vector and the frequency vector.

Parameters:
  • vmin – minimum value to consider for the dispersion diagram
  • vmax – maximum value to consider for the dispersion diagram
  • dv – velocity sampling
  • fmin – minimum frequency to consider
  • fmax – maximum frequency to consider
mute(xmute, tmute, key='tracl', ntaper=0, mode=0)[source]¶

Mute above or below a user-defined polygonal lineself.

Parameters:
  • key – SU header key
  • xmute – array of position values
  • tmute – array of time values
  • ntaper – number of points to taper before mute
  • mode – mute below (0) or above (1)
pfilter(freq, amps)[source]¶

Applies a zero-phase, sine-squared tapered filter (adapted from the sufilter command - Seismic Unix 44R1).

Parameters:
  • freq – array of filter frequencies (Hz)
  • amps – array of filter amplitudes
  • axis – time axis if dobs is a 2D array

taper(tr1=0, tr2=0, min=0.0, tbeg=0.0, tend=0.0, type='linear')[source]¶

Taper the edge traces of a data panel to zero.

Parameters:
  • dobs – input data to window
  • tr1 – number of traces to be tapered at beginning.
  • tr2 – number of traces to be tapered at end.
  • min – minimum amplitude to taper (<1., default=0.)
  • tbeg – length of taper (ms) at trace start (=0.).
  • tend – length of taper (ms) at trace end (=0).
  • taper – taper type: ‘linear’(default), ‘sine’, ‘cosine’
wiggle(clip=-1.0, key='tracl', label1=' ', label2=' ', title=' ', tracecolor='black', tracestyle='-', skip=1, xcur=1)[source]¶

Wiggle for SU files

Parameters:
  • clip – clip used to determine outside values to be clipped [-clip, clip]
  • key – header keyword (default tracl)
  • label1 – x-axis label
  • label2 – y-axis label
  • title – title of the image
  • tracecolor – color of the traces
  • tracestyle – style of the traces (‘–’, ‘:’, …)
  • skip – number of traces to skip for each plotted trace
  • xcur – factor to increase trace amplitudes on output
