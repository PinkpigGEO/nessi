update(**kwargs)[source]¶

Standard PSO update.

Parameters:
  • control – 0 for weight (default), 1 for constriction
  • c_0 – value of the control parameter (default 0.7298)
  • c_1 – value of the cognitive parameter (default 2.05)
  • c_2 – value of the social parameter (default 2.05)
  • topology – used topology (default ‘full’): full, ring, toroidal
  • ndim – number of particles in the first dimension if toroidal topology is used
