kill(key=' ', a=1, min=-1, count=1)[source]¶

Zero out traces. If min= is set it overrides selecting traces by header.

Parameters:
  • key – SU header keyword
  • a – header value identifying traces to kill
  • min – first trace to kill
  • count – number of traces to kill
pfilter(freq, amps, dt, axis=0)[source]¶

Applies a zero-phase, sine-squared tapered filter (adapted from the sufilter command - Seismic Unix 44R1).

Parameters:
  • freq – array of filter frequencies (Hz)
  • amps – array of filter amplitudes
  • dt – time sampling
  • axis – time axis if dobs is a 2D array

taper(tr1=0, tr2=0, min=0.0, tbeg=0.0, tend=0.0, type='linear')[source]¶

Taper the edge traces of a data panel to zero.

Parameters:
  • dobs – input data to window
  • tr1 – number of traces to be tapered at beginning.
  • tr2 – number of traces to be tapered at end.
  • min – minimum amplitude to taper (<1., default=0.)
  • tbeg – length of taper (ms) at trace start (=0.).
  • tend – length of taper (ms) at trace end (=0).
  • taper – taper type: ‘linear’(default), ‘sine’, ‘cosine’

  • tmax – maximum time to pass (=0)
